# OpsPilot

An agentic AI CLI tool that helps developers identify, analyze, and fix errors in their projects.

## Overview

OpsPilot is your intelligent companion for project maintenance and error resolution. It analyzes your codebase, detects runtime issues, and provides actionable insights to keep your projects running smoothly.

## Features

- **Automated Error Detection** - Scan your project for runtime issues and potential problems
- **Intelligent Analysis** - AI-powered incident analysis that understands your project context
- **Project-Aware** - Automatically detects project structure and configuration
- **Developer-Friendly** - Simple CLI interface with rich, informative output

## Installation

```bash
pip install opspilot
```

## Quick Start

Navigate to your project directory and run:

```bash
opspilot analyze
```

OpsPilot will analyze your project and help identify potential issues.

## Usage

```bash
opspilot [COMMAND]
```

### Commands

- `analyze` - Analyze the current project for runtime issues and errors

## Requirements

- Python 3.10 or higher

## How It Works

OpsPilot uses an agentic AI approach to understand your project structure, analyze runtime behavior, and provide intelligent recommendations for fixing errors. It adapts to your project's specific context and dependencies.
